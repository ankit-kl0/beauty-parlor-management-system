# Beauty Parlor Management System - Frontend

A React-based frontend for a **Beauty Parlor Management System** that allows clients to **book appointments**, choose services and staff, and allows admin to manage staff.

---

## Features

- **Service Selection:** Clients can browse and select from available services (hair, nails, skincare, etc.).
- **Staff Selection:** Choose available staff based on the selected service.
- **Appointment Booking:** Pick a date and book appointments easily.
- **Staff Management:** Admin can add, view, or remove staff.
- **Responsive Design:** Works on desktop and mobile devices.
- **API Integration:** Connects with backend APIs to fetch services, staff, and save appointments.
